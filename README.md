## PROJECT: SECOND-HAND VEHICLE PRICE PREDICTION MODEL 

### 1. Problem Statement 
In the Kenyan Second-hand vehicle market, subjetive pricing is a commong issue. There is no standard approach to pricing second-hand vehicles. Hence, prices tend to varry from one deaier to another and the difference may be significant. This calls for a data-driven solution that will provide transparency of prices.


### 2. Project Goal
Build a robust Machine Learning model (Price Prediction Model) that can help both buyers and sellers make informed decisions and reduce market inconsitences.
